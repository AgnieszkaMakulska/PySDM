# PySDM
Pparticle-based (a.k.a. super-droplet) simulations of atmospheric cloud microphysics.
The package features a Pythonic implementation of the Super-Droplet Method (SDM) Monte-Carlo algorithm, hence the name.
PySDM simulates the dynamics of population of particles immersed in moist air.
